# NAME

proctrace

# SYNOPSIS

**proctrace** \[**-h**\|**\--help**\] \[**-V**\|**\--version**\]
\<*subcommands*\>

# DESCRIPTION

# OPTIONS

**-h**, **\--help**

:   Print help

**-V**, **\--version**

:   Print version

# SUBCOMMANDS

proctrace-record(1)

:   Record the process lifecycle events from a command

proctrace-ingest(1)

:   Convert a raw recording into a processed recording that can be
    rendered

proctrace-render(1)

:   Render a recording in the specified display format
