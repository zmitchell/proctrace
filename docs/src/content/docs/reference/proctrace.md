# NAME

proctrace

# SYNOPSIS

**proctrace** \[**-h**\|**\--help**\] \[**-V**\|**\--version**\]
\<*subcommands*\>

# DESCRIPTION

# OPTIONS

**-h**, **\--help**

:   Print help

**-V**, **\--version**

:   Print version

# SUBCOMMANDS

proctrace-ingest(1)

:   Convert a raw recording into a processed recording such that it is
    ready for rendering

proctrace-sort(1)

:   Sort the output from a recording

proctrace-render(1)

:   Render the recording in the specified display format
